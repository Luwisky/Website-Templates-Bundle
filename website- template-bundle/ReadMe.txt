===================================
     WEBSITE TEMPLATE INSTRUCTIONS
===================================

Thank you for purchasing this template pack!  
This folder includes:  
1. Landing Page Template  
2. Portfolio Page Template  
3. Stylesheets (CSS) + Images  

Both templates are fully editable using any text editor.

-----------------------------------
 HOW TO USE THE TEMPLATES
-----------------------------------

1. Open the folder (Landing-Page OR Portfolio-Page).
2. Double-click the HTML file (index.html or portfolio.html) to preview it in your browser.
3. Open the HTML file in a text editor (Notepad, VS Code, Sublime, etc.).

-----------------------------------
 HOW TO EDIT TEXT
-----------------------------------

Inside the HTML file, you’ll see placeholders like:

    <h1>Your Business Name</h1>
    <p>Write your tagline here...</p>

Simply replace the placeholder text with your own content. Example:

    <h1>Monica Digital Agency</h1>
    <p>Helping businesses grow with technology</p>

Save the file, refresh your browser, and your changes will appear.

-----------------------------------
 HOW TO EDIT IMAGES
-----------------------------------

1. Open the "images" folder inside each template folder.
2. Replace the sample image files with your own images.
   (Tip: Keep the same file names OR update the image paths in the HTML.)

Example in HTML:

    <img src="images/your-photo.jpg" alt="Your Image">

Replace "your-photo.jpg" with the name of your own picture.

-----------------------------------
 HOW TO CHANGE COLORS
-----------------------------------

1. Open the CSS file (style.css or portfolio.css).
2. Find the color codes (like #4f46e5).
3. Replace them with your preferred hex code.

Example:

    background: #4f46e5;   →   background: #FF5733;

Save the file and refresh your browser to see the change.

-----------------------------------
 HOW TO EDIT THE RESUME BUTTON (Portfolio Only)
-----------------------------------

1. Replace "resume.pdf" with your own resume file.
2. Make sure the resume file is in the same folder as "portfolio.html".

    <a href="resume.pdf" download>Download Resume</a>

When clicked, it will download your resume.

-----------------------------------
 TROUBLESHOOTING
-----------------------------------

- If your changes don’t show, refresh the browser or clear cache.
- Make sure images are saved in the "images" folder.
- Always save the HTML/CSS files after editing.
